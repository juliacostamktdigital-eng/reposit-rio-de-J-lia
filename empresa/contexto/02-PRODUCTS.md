## Mapa do Portfolio (SWAS)

O portfolio da Colli & Co e organizado em quatro pilares. Cada produto pertence a um pilar e tem natureza one-time ou recorrente.

```
                        PORTFOLIO COLLI & CO
                               │
        ┌──────────────────────┼────────────────────────┐
        │                      │                         │                          │
     SABER                   TER                   EXECUTAR              POTENCIALIZAR
  (estrategia)           (ferramentas)             (operacao)            (amplificacao)
   one-time              one-time                  recorrente             recorrente
```

## Como usar este catalogo

- **Agente de vendas:** use para recomendar produtos por perfil de cliente e dor detectada.
- **Agente de diagnostico:** use para mapear o que o cliente ja tem vs o que ainda nao tem.
- **Agente de proposta:** use para montar combo de produtos com logica SWAS.
- **Agente de onboarding:** use para entender escopo de entrega a partir do produto contratado.

> Para detalhes de escopo, entregaveis e condicoes comerciais de cada produto, consulte a ficha em `empresa/produtos/<nome-do-produto>.md`.
