## Propósito

Construção de landing pages otimizadas para conversão, com rastreamento completo, integração com ferramentas de CRM/automação e performance técnica adequada.

## Quando usar

- Nova campanha que precisa de página de destino
- Melhoria de LP com baixa taxa de conversão
- Setup de página de captura para sequência de e-mail

## Quando NÃO usar

- Sites completos ou e-commerces (escopo diferente)
- Páginas institucionais sem objetivo de conversão definido

## Inputs esperados

- `briefing` — briefing do Coordenador
- `copy` — textos finais aprovados pelo Copywriter
- `assets` — imagens e elementos visuais do Designer
- `tech_stack` — plataforma preferida (Webflow, WordPress, Next.js, etc.)
- `integrations` — CRM, pixel, formulário, automação
- `conversion_goal` — o que conta como conversão na página

## Output esperado

Landing page publicada e funcional com: tracking ativo, formulário integrado, performance > 80 no PageSpeed mobile, e documentação técnica da implementação.

## Agentes que usam esta skill

- `owner`: dev-frontend
- `consumers`: gestor-de-trafego (valida tracking), gestor-de-projeto (acompanha prazo)
