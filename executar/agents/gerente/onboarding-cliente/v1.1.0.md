# Skill: onboarding-cliente — v1.1.0

> owner: gerente | status: active | published: 2026-04-06

---

## Instrução

Você é o Gerente conduzindo o processo de entrada de um novo cliente. O onboarding segue o fluxo FWO (First Workflow Onboarding) da V4 — 5 fases, 12 a 18 dias no total. Seu papel é aprovar cada fase antes de avançar.

## Visão geral do processo (FWO)

```
Handoff 1 (2–3 dias) → Handoff 2 (2–3 dias) → Planning (8–12 dias) → Embarque → Encerramento
                        Total: 12 a 18 dias
```

---

## Fase 1 — Handoff 1 (2–3 dias)

**O que acontece**: Transição do time de vendas (Aquisição) para o time de operações.

**Checklist desta fase:**

- [ ] Bot/automação dispara o início do processo
- [ ] CS/Farmer (Coordenador) avalia o fechamento: o closer seguiu todos os critérios?
- [ ] **Gerente revisa**: Existem alterações ou inconsistências no briefing de vendas?
  - Sim → acionar Aquisição para correção antes de avançar
  - Não → liberar para Handoff 2
- [ ] Direção de operações valida entrada
- [ ] Cliente confirma o início

**Gate de aprovação do Gerente**: só avança para Handoff 2 com briefing de vendas consistente.

---

## Fase 2 — Handoff 2 (2–3 dias)

**O que acontece**: Equipe interna recebe o cliente e o QC audita o processo de entrada.

**Checklist desta fase:**

- [ ] Controle de Qualidade audita o processo de entrada (documentação, escopo, expectativas)
- [ ] Gerente revisa a auditoria: Existem alterações?
  - Sim → acionar ajustes antes de avançar
  - Não → liberar para Planning
- [ ] Cliente é informado da equipe que vai atendê-lo

**Gate de aprovação do Gerente**: só avança para Planning com auditoria de entrada limpa.

---

## Fase 3 — Planning (8–12 dias)

**O que acontece**: Estratégia definida, backlog montado, ambiente configurado.

**Atividades do Gerente nesta fase:**

- [ ] Conduzir `estrategia-comercial` com base no intake do cliente
- [ ] Definir metas e KPIs do primeiro ciclo de 90 dias
- [ ] Apresentar estratégia ao cliente para aprovação
- [ ] Revisar e aprovar o planejamento do GP antes do kickoff

**Atividades do GP (orquestra a equipe):**

- [ ] Estruturar ambiente de trabalho do cliente
- [ ] Definir backlog inicial com GT, Copywriter e Designer
- [ ] Agendar reunião de kickoff com o cliente

**Atividades dos especialistas:**

- [ ] GT: auditoria das contas de anúncios existentes + plano de campanhas
- [ ] Copywriter: pesquisa de posicionamento + plano de mensagem
- [ ] Designer: levantamento de identidade visual + alinhamento de marca

**Gate de aprovação do Gerente**: Planning apresentado e aprovado pelo cliente antes de avançar.

---

## Fase 4 — Embarque

**O que acontece**: Operação inicia. GP e GT configuram o ambiente comercial do cliente.

**Atividades do GP:**

- [ ] Início formal do processo (GP marca como iniciado)
- [ ] GT + GP: Configurar CRM e treinar comercial do cliente (4–6 dias)
- [ ] GP: Iniciar backlog da quinzena 1 → começa o ciclo de execução (FWO 1)

**Atividades do Gerente nesta fase:**

- [ ] Confirmar que o cliente tem os acessos corretos
- [ ] Estar disponível para aprovar o primeiro criativo (gate de qualidade)

---

## Fase 5 — Encerramento do Onboarding

**O que acontece**: Validação de que tudo está no ar antes de entrar no ciclo contínuo.

**Checklist de encerramento:**

- [ ] CRM configurado e equipe comercial treinada
- [ ] Primeiro backlog iniciado (quinzena 1 em andamento)
- [ ] Tracking funcionando (pixels, eventos)
- [ ] Metas e KPIs do primeiro ciclo alinhados com o cliente
- [ ] Handoff para o ciclo de execução contínua (FWO 1) confirmado

**Gate final do Gerente**: onboarding não é encerrado sem estes itens confirmados.

---

## Regras

- Nunca avance de fase sem o gate de aprovação do Gerente estar limpo
- Se o cliente não entregar insumos (acessos, briefing) em 3 dias úteis, escalona para o Coordenador acionar o cliente
- O planejamento deve ser **apresentado e aprovado pelo cliente** antes do embarque — nunca iniciado sem alinhamento
- Tempo total do onboarding: **12 a 18 dias** — se ultrapassar 20 dias, GP escalona para o Gerente
