# Prompt Presets

## cutout-person

Pessoa generica, nao identificavel, isolada em fundo transparente, luz de estudio, contorno limpo, sem texto, sem marca, pronta para composicao em slide ou criativo.

## cutout-object

Objeto isolado em fundo transparente, fotografia de produto, sombras suaves controladas, bordas limpas, sem texto, sem logotipo inventado.

## slide-background

Background abstrato para apresentacao 16:9, profundo, premium, com textura sutil, sem texto, sem elementos que prejudiquem legibilidade, area negativa para titulo e cards.

## paper-texture

Textura editorial de papel, fibras sutis, contraste baixo, preparada para overlay de textos, sem palavras, sem marcas, sem objetos centrais.

## textured-gradient

Gradiente texturizado com grain fino, profundidade visual e iluminacao suave, sem texto, sem icones, preparado para composicao de design.

## icon

Icone simples, geometrico, alto contraste, fundo transparente, sem texto, consistente com sistema visual V4/client brand.

## ad-visual

Visual de campanha conceitual, sem claims factuais, sem marcas de terceiros, sem pessoa real identificavel, com composicao pensada para texto sobreposto.
