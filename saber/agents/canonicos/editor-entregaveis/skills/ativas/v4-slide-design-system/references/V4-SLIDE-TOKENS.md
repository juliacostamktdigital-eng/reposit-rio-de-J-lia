# V4 Slide Tokens

## Tamanho

- Aspect ratio: 16:9.
- PptxGenJS recomendado: `LAYOUT_16x9`.
- Tamanho equivalente: `10 x 5.625 in`.

## Cores

| Token | HEX | Uso |
|---|---|---|
| `v4.red` | `#E50914` | barra topo, headers, bullets, destaques |
| `v4.black` | `#1A1814` | fundo principal |
| `v4.white` | `#FFFFFF` | texto principal |
| `v4.gray` | `#464646` | texto secundario, bordas, apoio |
| `v4.yellow` | `#FFDD00` | CTA/destaque pontual |
| `v4.card` | `#111111` | cards sobre fundo escuro |
| `v4.cardAlt` | `#1E1E1E` | cards alternados |
| `v4.redDeep` | `#2A0005` | card premium/alerta |
| `v4.redLight` | `#FF8888` | insight/citacao |

## Tipografia

- Titulo: Morganite/Morgane Black, uppercase. Fallback: Arial Black.
- Corpo: IBM Plex Sans Regular. Fallback: Calibri.
- Labels: IBM Plex Sans SemiBold/Bold, uppercase.

## Layout base

- Barra superior vermelha: 1% a 2% da altura.
- Tag de secao: topo direito.
- Titulo: topo esquerdo, abaixo da barra.
- Logo: canto inferior direito, cerca de 3% a 5% da area visual.
- Margens minimas: 0.25 in nas bordas; 0.1 in para logo quando o layout permitir.

## Regras negativas

- Nao usar preto puro `#000000`.
- Nao usar vermelho incorreto `#E80403`.
- Nao usar paleta azul/verde/teal para categorias.
- Nao distorcer logo.
- Nao fazer slide full-image quando o conteudo precisa ser editavel.
