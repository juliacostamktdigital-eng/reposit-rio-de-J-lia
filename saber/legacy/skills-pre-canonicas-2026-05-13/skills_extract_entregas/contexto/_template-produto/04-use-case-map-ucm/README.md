# Use Case Map (UCM)

**O que entra aqui**

- Cenários em que o produto é a escolha certa (gatilhos, dores, estágio do funil).
- Combinações recomendadas com outros produtos e sequência típica.
- Anti-patterns: situações em que o produto costuma ser mal vendido ou mal absorvido.

**Status:** a preencher.
